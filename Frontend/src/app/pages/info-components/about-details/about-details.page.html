More About Me & Recipe Sage

A little about me

I'm a software developer in Long Beach, California.

I do a lot of work with nonprofits, and I spend a lot of time working on small projects.

At work, I develop EDU software for document annotation.


Why create Recipe Sage?

I created Recipe Sage to host recipes for me and my family. After trying multiple recipe programs online, I felt it would be best to create a recipe program that worked exactly as I wanted it to.

I'm a believer in freeware, and I was determined that Recipe Sage should be free. Please, feel free to suggest new features and improvements. I want Recipe Sage to grow and improve.

If you'd like to donate to support development, I'd really appreciate it. It costs money to run the servers, and I'd love help!
