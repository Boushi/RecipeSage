Download & Install

Click a platform for installation instructions

This Device Android Device

To install RecipeSage on Android:
1. Open the Chrome browser
2. Open the browser menu (the icon)
3. Click "Add to Home screen"
This will install RecipeSage as a native application on your device.

Apple Mobile Device

To install RecipeSage on Apple Mobile Devices (IOS):
1. Open Safari
2. Open the share menu (the icon)
3. Click "Add to Home Screen"
This will install RecipeSage as a native application on your device.

Desktop or Laptop Devices

To install RecipeSage on desktop or laptop devices:
1. Open Google Chrome browser
2. Open the browser menu (the icon)
3. Click "Install RecipeSage..."
This will install RecipeSage as a native application on your computer.
