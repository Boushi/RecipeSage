RecipeSage Browser Extension Tutorial

To view a full version of this tutorial, click here

The RecipeSage browser extension allows you to quickly snip recipes from the internet. It's fairly simple:
  1. Activate the RecipeSage clip tool by clicking the RecipeSage icon (you've already done this)
  2. Highlight some text on the page
  3. In the RecipeSage clip tool, press the scissors icon next to the corresponding field
  4. Edit the text as desired
  5. Repeat for all fields you want to clip
  6. Press the save button to save the recipe directly to your account
